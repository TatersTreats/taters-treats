# Tater's Treats SEO foundation

This package is a complete SEO-structured foundation for the Tater's Treats site.

## Included
- index.html
- dogbowl-checkout-success.html
- package.json
- api/create-dogbowl-checkout-session.js
- assets/css/taters-treats-premium-small-batch-dog-treats.css
- assets/javascript/taters-treats-dogbowl-shopping-experience.js
- assets/images/branding
- assets/images/hero
- assets/images/products
- assets/images/story
- assets/images/dogbowl

## Notes
- Upload these contents as a full overwrite/update to the existing GitHub repo root.
- Do not wipe the repo again.
- Stripe checkout uses the existing price IDs.
- Ensure Vercel has STRIPE_SECRET_KEY set in Environment Variables.
